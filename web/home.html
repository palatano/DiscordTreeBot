Music Commands

Playing music is as easy as joining voice and typing:

For more information, click on the button below.

Use an assortment of analysis utilities, such as searching on YouTube and Google.

;search [query]

For more information, click on the button below.

Miscellaneous Commands

Use various commands for entertainment or information.

;bot

For more information, click on the button below.

Basic commands:
◉ ;add [song] - Adds the song with either a keyword search or a direct URL.
◉ ;skip - Skip the current song.
◉ ;req [song] - Allow a non-authorized user to request a song and an authorized user to add it.
◉ ;np - Displays the current song that is playing.

Advanced commands:
◉ ;pause - Pause the song.
◉ ;list - Returns the current song list. Turn on/off automatic playlist posting with ;list [on|off]
◉ ;unpause - Unpause the player.
◉ ;leave - Leave the music channel.
◉ ;join - Join the music channel.
◉ ;musichelp - Music command list.
◉ ;cnl - Cancels the current menu, if open.
◉ ;undo - Undo the last song added.

Basic commands:
◉;info [name]: Returns your join date to the server. If you have left/been kicked the join date will be the day you rejoined.
◉;uniqueusers [date]: Gives an excel file with the unique user name, time stamp, and message.
◉;search [search]: Returns the first three search results from a google search query.
◉;ping: Returns the latency of the bot.
◉;youtube [search]: Returns the first three search results from Youtube.

Miscellaneous Commands
◉;test: Tests if the bot is online with a :joy: emoji.
◉;help: Help menu.
◉;bot: Gives the bot information.
